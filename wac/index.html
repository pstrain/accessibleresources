Web Accessibility Checker

Evaluate websites against the industry-standard accessibility guidelines using the latest tools.

Step by step guide to using Web Accessibility Checker

1. Create new project with a base URL.

2. Select page(s) - save one page or save all pages linked from a page.

3. Audit - tap start to start automatically auditing all the pages in your list

4. Review results.

FAQ

What is a batch?

A 'batch' is a collection of results. You can review the same set of pages at a later date, keeping the results separate to review progress.
